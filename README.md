# Algorithm Visualizer

A Python project that demonstrates core Data Structures & Algorithms by visualizing sorting algorithms and stepping through graph algorithms.

## Features

### Sorting Visualizer (Matplotlib)
- Bubble Sort
- Merge Sort
- Quick Sort
- Animated bar-chart visualization

### Graph Algorithms (Console Steps)
- BFS (Breadth-First Search)
- DFS (Depth-First Search)
- Dijkstra (Shortest Path, weighted graph)

## Project Structure
- `main.py` — entry point
- `src/sorting_visualizer.py` — sorting visualization
- `src/graph_algorithms.py` — BFS/DFS/Dijkstra
- `requirements.txt` — dependencies

## Requirements
- Python 3.x
- matplotlib
